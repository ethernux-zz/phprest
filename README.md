# PHPRest



PHPRest is a simple RestAPI in PHP who cans manage products and categories.
This a basic version 


#  Features!

  - Creating products
  - Updating products
  - Deleting products
  - Searching batch or single product
  - Searching by some tags ex:  name,descr,price,category etc


# Requirements
 - PHP 5.1 or early
 - MySQL any version
